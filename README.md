  # Système de recommandation de PC
  
  ## Présentation
  Ce projet est une application web interactive de recommandation d'ordinateurs développée dans le cadre d’une formation en Data Analyst.
  
  L’objectif est d’aider les utilisateurs à choisir un PC adapté à leurs besoins réels et à leur budget, en s’appuyant sur une analyse des caractéristiques techniques et des performances.
  
  ---
  
  ## Fonctionnalités
  
  - Recommandations personnalisées  
    Propose des ordinateurs en fonction de l’usage, du budget et des préférences utilisateur.
  
  - Analyse des caractéristiques techniques  
    Prend en compte des critères tels que la RAM, le processeur, le stockage et le système d’exploitation.
  
  - Scraping de données  
    Collecte des données depuis des sites comme LDLC.com pour obtenir des informations actualisées sur les produits.
  
  - Enrichissement par benchmarks  
    Intègre des données de performance pour processeurs et cartes graphiques afin d’évaluer objectivement les configurations.
  
  - Application web interactive  
    Développée avec Streamlit pour fournir une interface simple et accessible.
  
  - Assistance IA  
    Utilisation de Gemini pour améliorer l’expérience utilisateur et l’aide à la décision.
  
  ---
  
  ## Stack technique
  
  - Python  
  - Scraping et nettoyage de données  
  - Streamlit  
  - Gemini (IA)  
  - Bases de données  
  - Benchmarks de performance  
  
  ---
  
  ## Fonctionnement
  
  1. L’utilisateur renseigne ses besoins (usage, budget, préférences).  
  2. Les données produits sont récupérées via scraping.  
  3. Les performances sont enrichies grâce à des benchmarks.  
  4. Les configurations sont analysées et filtrées.  
  5. Le système propose des recommandations adaptées.
  
  ---
  
  ## Application
  
  View on App:  
  https://recommandation-pc.streamlit.app

  # PC Recommendation System
  
  ## Overview
  This project is an interactive web application for computer recommendations, developed as part of a Data Analyst training program.
  
  The goal is to help users choose a PC that matches their real needs and budget, based on an analysis of technical specifications and performance.
  
  ---
  
  ## Features
  
  - Personalized Recommendations  
    Suggests computers based on usage, budget, and user preferences.
  
  - Technical Specifications Analysis  
    Takes into account criteria such as RAM, processor, storage, and operating system.
  
  - Data Scraping  
    Collects data from websites such as LDLC.com to obtain up-to-date product information.
  
  - Benchmark Enrichment  
    Integrates performance data for processors and graphics cards to objectively evaluate configurations.
  
  - Interactive Web Application  
    Built with Streamlit to provide a simple and user-friendly interface.
  
  - AI Assistance  
    Uses Gemini to enhance user experience and decision-making support.
  
  ---
  
  ## Tech Stack
  
  - Python  
  - Data scraping and cleaning  
  - Streamlit  
  - Gemini (AI)  
  - Databases  
  - Performance benchmarks  
  
  ---
  
  ## How It Works
  
  1. The user inputs their needs (usage, budget, preferences).  
  2. Product data is collected via scraping.  
  3. Performance data is enriched using benchmarks.  
  4. Configurations are analyzed and filtered.  
  5. The system generates relevant recommendations.

  ## Application
  
  View on App:  
  https://recommandation-pc.streamlit.app
